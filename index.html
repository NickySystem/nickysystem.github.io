Garden

Electronic Music Performance Tool

WIP A (currently) Python / PD based system for live music performance. Has themes and MIDI output.



HEDJAM

The Generative Virtual Band

HEDJAM puts the performer in the role of player character. Dialogue with virtual band members drives a procedural music engine. Good for streaming and projection gigs. Fun to play with.


Download it on itch.io here.


Tiny Techno Psuedo Language

A tiny live coding REPL for minimal techno

A Pure Data based command-line live coding system. Contains a 5 piece drum machine, bass synth & melody synth. Built for minimal techno sets. (◕‿◕)

Download it on github here.



UnSync

An Asynchronous MIDI Sequencer

UnSync uses collision events to trigger MIDI output. Make sure you have a device hooked up before using.


Play it on itch.io here.
